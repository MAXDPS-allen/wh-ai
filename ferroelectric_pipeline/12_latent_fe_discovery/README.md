# 12 · 潜在铁电发现 (Latent Ferroelectric Discovery) — 方法学创新

> 方法学论文的核心模块（方向 A+B）。

## 想法

把铁电发现**反过来**做：不再"给定极性结构判断是否铁电"，而是从**高对称非极性母相
单独**预测
- (A) 是否存在极性失稳 + 失稳本征矢（极性软模，原子位移场，E(3)-等变）；
- (B) 由此产生的(假想)极性相自发极化 \(P_s\)（经 Born 电荷路径，规避极化量子歧义）。

**无需**计算极性结构、声子或 Berry 相 → 可在海量稳定非极性晶体中筛选"隐藏的"铁电体。

物理依据：位移型铁电的微观起源就是高对称相在 Γ 的极性 TO 软模失稳（Cochran 软模理论）。
新颖性（已检索确认）：现有工作把畸变模式当**输入描述符**或预测**已知结构**的电响应；
本方法把失稳本征矢当**等变输出**并用于**生成**未知极性相 → 反演工作流。

## 关键发现（诚实结果）

从零训练（Smidt 204 训练 / 51 验证）：

| 指标 | 结果 | 基线 |
|------|------|------|
| mode 方向余弦 (验证) | **0.10** | 随机 (3N 维) ≈ 0.10 |
| mode 方向余弦 (**过拟合 6 个**) | **0.68** | —— |
| Ps 矢量余弦 (验证) | 0.55 | 随机(3D) 0.50 |
| Ps R² (验证) | −0.40 | —— |

**诊断**：过拟合测试达 0.68（远高于随机）证明**模型能学、无 bug**；但从 204 个**化学多样**
的材料**无法泛化**（验证集=随机）。根因是**数据稀缺**——~200 个差异巨大的材料不足以学习
3N 维等变矢量场。这与模块 10 一致（同等数据量下标量带隙可学、几何性 Ps 不可学）。

## 这恰好指向真正的方法学创新：等变迁移学习

数据瓶颈的解法本身就是更强的方法贡献：

1. **声子本征矢预训练**：在大规模声子数据库（MP phonon DB / Petretto 等 ~1.5k+ 材料，
   含 Γ 本征矢）上预训练等变模型——极性软模只是其特例。再在极性失稳上微调。
2. **Born 电荷预训练**（B 的量子无关路径）：用已有电响应数据集预训练 Z* 头。
3. **主动学习扩库**：用本仓库 `09_dft_validation` 的 DFT 闭环不断扩充极性失稳标注。

→ 论文叙事升级为 **“equivariant transfer learning from phonons to ferroelectric soft modes
for latent-ferroelectric discovery”**，比从零学习更有分量，且解决了数据瓶颈。

## 迁移学习实现（已完成基础设施）

数据瓶颈的解法已落地。MP 声子库**无 Γ 本征矢字段**，但有 **Born 有效电荷 Z***
（14,565 材料）—— 这正是 B 头(量子无关极化)的预训练目标，也让等变主干学到丰富的局域
电响应表示供下游迁移。

| 组件 | 文件 | 说明 |
|------|------|------|
| Born 预训练集 | `build_born_dataset.py` | MP 声子库 → 10,403 材料 (structure, Z*), nsites≤20 |
| Z* 等变头 | `model.py` `head_zstar` | 对称 Born 电荷 (0e+2e, CartesianTensor `ij=ji`) |
| 预训练 | `pretrain.py` | 在 Born 电荷上训练等变主干 |
| 微调 | `finetune.py` | 加载预训练主干, 在 Smidt mode+Ps 上微调 (先冻结后解冻) |

### 实测状态

- **Born 预训练**：Z* R²=0.20（小模型、2.5k 子集、CPU、~10 epoch）。主干**确实在学**
  等变电响应；更大模型/全量 1 万材料/更多 epoch 可显著提升（需 GPU）。
- **迁移微调**：管线已跑通并在运行，但 **CPU 是瓶颈**（e3nn ~2-3 min/epoch，多用户争抢），
  收敛实验（数百 epoch）在交互式会话中不可行。

### GPU 基础设施（已解决）

g 计算节点是 **glibc 2.17 (CentOS 7)**，与现代 pip wheel (numpy/torch 需 glibc≥2.27) 不兼容。
解法: 用 **conda-forge** 建 `fe_gpu` 环境 (conda-forge 面向 glibc 2.17), CUDA 11.8 pytorch +
pip e3nn。需 `CONDA_OVERRIDE_CUDA=11.8` (登录节点无 GPU) + 运行时 `LD_LIBRARY_PATH=$CONDA_PREFIX/lib`
(补 libstdc++)。g 节点 GPU 训练比登录节点 CPU **快 ~50×**。

### GPU 实测结果（`gpu_results.json`）

| 实验 | 结果 | 结论 |
|------|------|------|
| **Born 电荷预训练** (1.04 万材料) | **Z\* R² = 0.952** | ✅ 等变 Born 电荷模型很强 |
| 软模迁移 (从 Born 主干微调) | mode_cos 0.10→**0.18** (随机 0.10) | ⚠️ 仅边际提升 |
| 直接预测 Ps (从母相) | R² 仍为负 | ⚠️ 不工作 |
| Born 路径 Ps (预测 Z*·真实模式) | 与 DFT Ps 相关 **0.33** | ⚠️ 线性近似对大畸变失效 |

### 诚实结论与诊断

- **成功**: 等变 Born 电荷预测在 1 万材料上 R²=0.95 —— B 路径(量子无关极化)的基石可学。
- **未成功**: 从高对称母相预测**极性软模/Ps**仍接近随机, 即使有 Born 预训练。
  **根因**: Born 电荷编码的是"位移→极化"的线性电响应, 而软模是"哪个畸变会失稳"——
  属于**晶格不稳定性 (力常数/Hessian)** 的信息, Born 电荷不包含。且 Ps 对大畸变需
  **沿路径积分 Z\***, 而非母相处线性 Z\*·u。

### 真正缺的拼图: 声子力常数 / 本征矢数据

软模发现头需要**声子力常数 / Γ 本征矢**作训练信号 (MP 声子库 summary 未暴露此字段)。
下一步:
1. 从 MP 拉取 phonopy 力常数对象 (S3) 或换用含本征矢的声子数据集 → 预训练软模头。
2. 或直接预训练**动力学矩阵/Hessian** (等变 rank-2 对), 软模 = 最小本征矢。
3. Ps 用沿路径多点 Z\* 积分 (而非线性近似)。

> 当前阶段诚实定位: GPU 管线打通、Born 电荷头很强 (可独立成文: "等变 Born 电荷→量子无关
> 极化"), 但"从母相发现潜在铁电"的软模头需声子力常数数据才能突破。

### 软模数据获取尝试 (Γ 本征位移) — 受 SSL 阻塞

已定位正确数据与接口: `get_phonon_bandstructure_by_material_id` 返回 `eigendisplacements`
(nbands, nq, natoms, 3), 取 Γ 点即软模本征矢。已建好:
- `fetch_phonon_modes.py` — 顺序拉取 + 增量保存 + 每次 240/600s 超时 + 40 次 SSL 重试 (可断点续传)
- `pretrain_modes.py` — 用 Γ 最软光学模本征位移预训练 **mode 头** (与下游同任务的正确迁移)

**障碍**: 本集群到 `api.materialsproject.org` 的 **SSL 极不稳定** (`SSLEOFError`,
成功率在分钟级内 0%↔100% 漂移)。单条拉取在 SSL 好的窗口 ~15s 成功, 坏窗口 40 次重试仍失败。
→ 逐条 API 拉取 ~800 材料不可靠/极慢, 当前后台缓慢累积中 (resumable)。

**已执行的稳健替代 (成功绕开 flaky API)**:
- 诊断: `curl` 到 MP 5/5 稳定 → 问题在 Python `mp_api` 长连接 SSL 失效 (并发更糟)。
- 解法: 从 **Figshare** 批量下载 **Petretto 2018 DFPT 声子库** (1521 材料):
  `phonon.tar.bz2` (频率+不稳定标志+Born电荷) + `ddbs.tar.bz2` (296MB, ABINIT DDB)。
- 装 `abi` 环境 (conda-forge abipy+abinit), 用 **anaddb 本地**对 1521 个 DDB 算 Γ 声子
  本征矢 (`extract_ddb_modes.py`): **1534 材料, 0 错误, 全本地无网络**。

### 最终结果与决定性发现

| 模型 | 软模本征矢 val cos |
|------|-------------------|
| 从零 | 0.10 |
| Born 迁移 | 0.18 |
| **声子软模迁移 (1534 真本征矢)** | 0.17 |
| 随机基线 (3N 维) | ~0.13 |

**决定性发现 (诚实)**: 直接预测软模**本征矢方向**是**病态问题**——**72% 的材料其最软光学模
(近)简并** (Γ 点对称简并), 单一本征矢在简并子空间内任意, 模型无法预测任意选择 → 无论
数据量/预训练如何, val cos 都停在随机附近。**这不是数据问题** (已有 1534 个真本征矢), 是
**目标定义问题**。

### 原则性修正 (验证过的下一步)

预测**动力学矩阵 / Hessian** (等变 rank-2 每原子对张量, 简并无关、良定义), 软模 = 其最小
本征对。数据已在手 (1534 个 DDB 含完整力常数, 可由 anaddb 直接给出 Hessian)。这绕开了
本征矢简并的病态性, 是该方法学路线的正确架构。

### 可独立成文的成果

**等变 Born 电荷 → 量子无关自发极化** (Z\* R²=0.95, 1 万材料) + 极化量子洞见 (193→9.9)。

## 级联 (Cascade) 尝试: 简单分类初筛 + 精细回归

(用户建议) 既然本征矢回归病态, 改"分级": Tier 1 简单分类初筛, Tier 2 精细回归。

| 级 | 任务 | 实现 | 结果 |
|----|------|------|------|
| **Tier 1** | 是否有 Γ 软模 (潜在铁电/动力学失稳) 分类 | 等变标量 logit, BCE+pos_weight (`train_instability.py`) | ✅ **AUC=0.77, balAcc=0.70, top-10% 富集 4.8×** |
| **Tier 2** | 简并无关的每原子软模参与张量 S_i=Σ e_i⊗e_i (rank-2) | 等变 rank-2 头 (`train_tier2.py`) | ⚠️ 张量余弦 0.50 < **均值基线 0.667**, participation R²<0 |

**Tier 1 成功且有用**: 在 1534 声子材料 (8.9% 失稳) 上 AUC 0.77, 取打分前 10% 即得
~43% 命中 (基率 9%) → 可作为"在海量非极性晶体中初筛潜在铁电"的**判别器**。

**Tier 2 仍不收敛**: 即便用简并无关的参与张量 + Born 主干热启动, 精细模式形状仍学不过
"预测均值张量"基线。

### 贯穿所有实验的诚实规律

| 任务类型 | 可学? | 例 |
|----------|-------|-----|
| 粗粒度/标量/分类 | ✅ | 失稳分类 AUC 0.77; 带隙 R² 0.6-0.7; Born 电荷 Z\* R² 0.95; 可切换性 AUC 0.8 |
| 精细几何/方向 | ❌ | Ps 矢量; 软模本征矢 (cos≈随机); 参与张量 (< 基线) |

→ ~200-1500 个**多样**材料不足以学精细几何量。**级联 Tier-1 初筛器是当前可用的发现工具**;
Tier-2 精细回归需 (a) 主动学习用 `09_dft_validation` 闭环扩库, 或 (b) 按化学族/结构原型分治。

> 完整数字见 `cascade_results.json` / `methods_results.json`。

## 级联端到端验证 (Tier-1 筛选 → DFT 确认) ✅

**Tier-2 重构成功**: 把"精细回归"从(病态的)本征矢方向改为**标量软度**(最软光学模带符号频率):
`train_tier2_freq.py` → **R²=0.45, MAE=1.28 THz, sign_AUC=0.87** (作为分类器甚至优于 Tier-1)。

**Tier-1 用于发现 + DFT 验证** (`screen_tier1.py`, `verify_phonon.py`):
1. 在 200 个 MP 非极性稳定绝缘体(排除 1534 训练集)上打分 → top 候选为**双钙钛矿**
   Ba₂(Ca/Dy/Er)(Mo/W/Nb/Sb)O₆ + BCl₃ (物理上正是软模高发体系)。
2. 对 2 个高分 + 1 个低分对照做**第一性原理 Γ 声子**验证:

| 候选 | Tier-1 | DFT 最小频率 | DFT 判定 | 一致 |
|------|--------|------------|---------|------|
| **Ba₂CaMoO₆** (mp-19403) | 失稳(1.0) | **−3.47 THz** | 失稳 | ✅ |
| BCl₃ (mp-23184) | 失稳(1.0) | −0.14 THz | 失稳(边际)| ✅ |
| Ac₂O₃ (mp-11107, 对照) | 稳定(0.0) | **+0.21 THz** | 稳定 | ✅ |

→ **DFT 确认 Tier-1 3/3**。双钙钛矿 **Ba₂CaMoO₆ 被第一性原理确认为潜在铁电**(Γ 软模 −3.47 THz,
其立方相不稳, 将畸变到极性相)。这验证了"等变初筛器 → DFT 确认"的发现闭环。

> 注: 分数饱和在 1.0 (类不平衡 pos_weight 致过自信), BCl₃ 边际 (2×2×2 给 +0.05, 1×1×1 给 −0.14);
> 但强真阳 (钙钛矿) + 对照稳定的方向判别清晰。规模化筛选 + DFT 主动学习可进一步提升。
