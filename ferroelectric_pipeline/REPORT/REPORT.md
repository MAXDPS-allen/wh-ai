# 机器学习 + 第一性原理铁电材料发现：方法与结果报告

**用途**：计算资源申请支撑材料 · **代码**：https://github.com/MAXDPS-allen/wh-ai （`ferroelectric_pipeline/`）

---

## 摘要

我们建立了一套**机器学习驱动、第一性原理闭环验证**的铁电材料发现系统，并已取得可验证的
阶段性成果：

1. **完整的第一性原理铁电验证管线**（复刻 Smidt et al. *Sci. Data* 2020 工作流），对一个
   候选材料 **SrAlGeH** 给出了完整铁电证明：自发极化、能量双势阱、切换势垒、声子稳定性、
   杂化泛函带隙（图 3–4）。
2. **等变神经网络级联**用于"潜在铁电"发现：在 1,534 个声子材料上训练，从结构判别动力学
   失稳（软模）。**ML 预测经第一性原理 Γ 声子验证，3/3 全部确认**（图 1c）。
3. **大规模筛选**：对 1,500 个 Materials Project 非极性稳定绝缘体打分，产出 **173 个潜在
   铁电候选**清单（图 2），涵盖氟化物/氯化物 elpasolite、Dion-Jacobson 层状钙钛矿等家族。

**当前瓶颈是算力**：候选验证依赖高通量 DFT（每个候选数十个 VASP 计算），且共享节点经常
被其他用户占满。申请专用 GPU 资源即可把 173 候选（及全 MP 规模化筛选）推进到完整验证 +
主动学习闭环。

## 1. 方法

### 1.1 第一性原理铁电验证管线（`09_dft_validation/`）
对任意极性候选，自动完成：找高对称非极性母相 → 非极性↔极性 8 点插值 → VASP 弛豫/静态/
Berry 相极化（`LCALCPOL`）→ 极化分支跟踪（含极化量子约化）→ 声子有限位移（phonopy）→
切换势垒（NEB 路径）→ HSE06 带隙。集群无 SLURM，自研 SSH 多节点 GPU 调度器（含失败重试、
节点能力探测）。

### 1.2 等变级联发现模型（`12_latent_fe_discovery/`）
**关键洞见**：直接回归软模本征矢是**病态问题**（72% 材料软模简并 → 方向不可学）。改用**级联**：
- **Tier-1**（等变分类器）：从结构判别"是否存在 Γ 软模（潜在铁电/动力学失稳）"。
- **Tier-2**（标量回归）：预测最软光学模的带符号频率（失稳强度），用于精排。
- **主干预训练**：在 10,403 个材料上预训练 E(3) 等变 Born 有效电荷张量预测（Z\* R²=0.95），
  作为表示学习主干。
- **训练数据**：Petretto 等 1,534 个声子材料的 Γ 本征矢/频率（经 ABINIT/anaddb 本地解析 DDB
  获得，绕开不稳定的 MP API）。

## 2. 结果（均已验证）

### 2.1 单材料完整铁电证明：SrAlGeH（图 3、4）
| 判据 | 结果 | 方法 |
|------|------|------|
| 自发极化 Pₛ | **9.9 μC/cm²**（沿极性 c 轴）| Berry 相 + 分支跟踪 + 量子约化 |
| 能量双势阱 | **74 meV/atom**（极性相为基态）| DFT |
| 切换势垒 | **0.296 eV/f.u.**（≈ LiNbO₃）| +P→鞍点→−P |
| 带隙 (PBE→HSE06) | 0.69 → **1.48 eV** | 杂化泛函 |
| 动力学稳定 | **无虚频**（min +0.18 THz）| phonopy + VASP |

→ 六项决定性判据全部满足，SrAlGeH 是真实、稳定、可切换的铁电体。

### 2.2 级联模型性能与稳定性（图 1a, 1b, 6a）

**5-折交叉验证**（N=1534，mean ± std）——证明指标稳定、非单次划分侥幸：

| 模型 / 指标 | 5-折 CV |
|------|------|
| Tier-1 失稳分类 **AUC** | **0.81 ± 0.06** |
| Tier-1 top-10% **富集** | **5.1 ± 1.0×**（基率 9%）|
| Tier-2 **sign-AUC**（失稳判别）| **0.82 ± 0.03**（很稳定）|
| Tier-2 R²（绝对软度回归）| 0.24 ± 0.13（对划分敏感，见下）|
| Born 电荷 Z\*（主干）| R² = **0.95**（10,403 材料）|

**等变性验证**：随机旋转输入 → 标量输出不变（误差 ~2×10⁻⁵）、矢量输出随旋转（误差 ~1×10⁻⁴）。
确认模型严格满足 E(3) 对称性（数值精度级误差），这是物理正确性与泛化稳定性的硬保证。

诚实记录：**判别能力（AUC / sign-AUC）跨折稳定**且强；**绝对软度回归 R² 噪声大**（0.1–0.44，
对划分敏感）—— 这是 ~1,500 个多样材料的**数据量瓶颈**，正是主动学习闭环（DFT 回流扩库）
要解决的，也是申请算力的核心动机之一。

### 2.2.1 推理 vs 计算验证的时间成本（图 6b）—— 申请算力的核心论据

| 环节 | 单材料耗时 | 说明 |
|------|-----------|------|
| **ML 推理**（Tier-1+Tier-2 打分）| **~15 ms**（CPU）| 含前向；GPU 更快 |
| **DFT 验证**（完整）| **~3.2 小时** | 实测：SrAlGeH 完整证明 29 个 VASP 计算/6.3 h；Ba₂CaMoO₆ 验证 2.7 h；Ac₂O₃ 0.6 h |

→ **ML 比 DFT 快约 8×10⁵ 倍**。这正是价值所在：ML 以毫秒级在**数千–数万**个候选中初筛，
DFT 只对最有希望的少数做小时级确认。**算力投到 DFT 验证端，由 ML 精准导向，单位算力发现
效率最大化。**

### 2.3 ML→DFT 发现闭环验证（图 1c）
对级联筛出的候选做第一性原理 Γ 声子验证：

| 候选 | ML 预测 | DFT 最小频率 | 一致 |
|------|---------|------------|------|
| **Ba₂CaMoO₆**（双钙钛矿）| 失稳 (100%) | **−3.47 THz** | ✅ |
| BCl₃ | 失稳 (100%) | −0.14 THz | ✅ |
| Ac₂O₃（对照）| 稳定 (0%) | +0.21 THz | ✅ |

→ **DFT 3/3 确认 ML 预测**。双钙钛矿 Ba₂CaMoO₆ 经第一性原理确认为潜在铁电。

### 2.4 发现版图（图 2）
1,500 个 MP 非极性稳定绝缘体 → **173 个潜在铁电候选**，家族包括：双钙钛矿氧化物 (35)、
氟化物 elpasolite K₂NaMnF₆/KRb₂MF₆ (26)、氯化物 elpasolite (19)、Dion-Jacobson 层状钙钛矿
CsSr₂Nb₃O₁₀ (2)、其它/新颖 (91)。完整清单见 `discovery_list.json`。

## 4. 算力使用与申请理由

**已用算力**：集群 GPU 节点 g1–g9（VASP 6.4.2 CUDA）。已自研无调度器的 SSH 多节点 GPU 派发，
并解决多项工程问题（节点 AVX512 兼容性、MPI/库路径、LCALCPOL 的 CPU 后端、失败重试）。

**瓶颈（即申请理由）**：
1. **高通量 DFT 验证**：173 个候选每个需 ~20–40 个 VASP 计算（弛豫/声子/极化/势垒/HSE）。
   完整验证 173 候选 ≈ 数千个 VASP 作业。
2. **共享节点拥挤**：验证经常因其他用户占满节点而拖慢甚至停滞（已实测：单材料因节点超订
   收敛速度下降 ~10×）。**专用 GPU 配额是把发现清单推进到验证的关键**。
3. **主动学习闭环**：DFT 验证结果回流扩充训练集 → 重训模型攻克方向预测（当前数据瓶颈），
   需要持续算力支撑迭代。
4. **规模化筛选**：当前仅筛 1,500，MP 非极性晶体有数万个；全量筛选 + top 候选验证需更多算力。

**申请规模建议**：专用多 GPU 配额（用于并行高通量 VASP）+ 持续运行权限，以完成 173 候选的
完整第一性原理铁电证明并启动主动学习闭环。每确认一个新铁电体即为可发表的计算发现。
